
# WebSocket Chat Application

This is a real-time chat application built using JavaScript and WebSocket.

## Features
- Real-time messaging
- WebSocket server
- Multiple users support

## Technologies
- JavaScript
- Node.js
- WebSocket (ws)

## How to Run
1. Install Node.js
2. Run `npm install ws`
3. Start server: `node server.js`
4. Open index.html in browser
